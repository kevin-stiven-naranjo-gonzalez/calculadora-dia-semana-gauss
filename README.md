# Calculadora Día de la Semana

Calcula el día de la semana usando la fórmula de Gauss (A+B-C+D).
fuente --> https://en.wikipedia.org/wiki/Determination_of_the_day_of_the_week#:~:text=El%20algoritmo%20de%20Gauss%5Beditar

 Schwerdtfeger, Berndt E. (2010-05-07). "Gauss' calendar formula for the day of the week" (PDF) (1.4.26 ed.). Retrieved 2012-12-23. 


## Uso
desde el terminal:
python calculo_día_semana_zeller.py


## Requisitos
- Python 3.x
- tkinter (incluido en Python estándar)
